# M0 lx-music-mobile 在线生态移植 — 核心闭环 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 LynMusic_Kotlin（KMP + Compose Multiplatform）上跑通"搜索→播放→歌词随动→收藏→重开再播"的在线音乐核心闭环，覆盖 Android / iOS / macOS / Windows / Linux 五平台。

**Architecture:** 新增两个 Gradle 子模块 `shared:scripting`（三平台 JS 引擎抽象 + Host Bridge）和 `shared:online`（领域层：MusicSourceFacade + JsMusicSource + Repository + Store + Screen）。六源（kw/kg/tx/wy/mg；xm 为 lx 侧 stub，M0 标 disabled）的 lx 原版 JS 经 Node+esbuild 改造（shim `lyn.crypto/lyn.buffer/lyn.request/lyn.base64/lyn.zlib/lyn.iconv/lyn.timer/lyn.nativeCrypto`）后作为每源独立 bundled `.js` 资产打进 `composeResources/files/sdk/`。`SongUrlResolver` 在现有 `PlaybackRepository.loadGatewaySafely()` 内懒加载 URL，三平台 Gateway actual 不动。

**Tech Stack:**
- Kotlin 2.3.20 / Compose Multiplatform 1.11.0-beta01 / Ktor Client 3.4.2 / Room 2.8.4 / kotlinx-coroutines 1.10.2 / JDK 21
- Android JS 引擎：`com.whl.quickjs.wrapper:quickjs-android`（WeChat 维护，替换 spec 原选型的停维 `com.quickjs:quickjs-android`）
- JVM JS 引擎：`org.graalvm.polyglot:js-community`（纯解释器，冷启动友好）
- Apple JS 引擎：系统 `JavaScriptCore.framework` + Kotlin/Native cinterop（iOS/macOS 共用一个 `.def`）
- 资产管线：Node.js + esbuild（开发机工具链，产物 check-in git）

## Spec 引用与修正

本 plan 基于 `docs/superpowers/specs/2026-04-17-lx-online-ecosystem-port-design.md`。以下为调研后对 spec 的修正（spec 文档不变）：

| 原 spec | 实际采用 |
|---|---|
| 六源全部接通 | **M0 接通 5 源**（kw/kg/tx/wy/mg）；xm 在 `SourceManifest` 标 disabled 并保留槽位 |
| Android `com.quickjs:quickjs-android` | `com.whl.quickjs.wrapper:quickjs-android`（WeChat 版） |
| JVM `org.graalvm.js:js` | `org.graalvm.polyglot:js-community`（纯解释器） |
| 音质：`sq → hq → 128k` 降级 | 按 lx 真实枚举 `flac24bit → flac → 320k → 192k → 128k` |
| shim 仅 md5/buffer/base64 | 额外 shim `pako`(inflate) / `iconv-lite`(decode) / `react-native-background-timer` / `@/utils/nativeModules/crypto`(AES/RSA/SHA1) |
| `findMusic` 10-pass 完整算法 | **M0 简化为 2-pass**（精确 name+singer；放宽 duration 容差 3s）；完整算法延到 M1/M2 |

```kotlin
package top.iwesley.lyn.music.scripting

sealed class JsValue {
    data object Null : JsValue()
    data object Undefined : JsValue()
    data class Bool(val value: Boolean) : JsValue()
    data class Num(val value: Double) : JsValue()
    data class Str(val value: String) : JsValue()
    data class Arr(val items: List<JsValue>) : JsValue()
    data class Obj(val entries: Map<String, JsValue>) : JsValue()
    data class Bytes(val data: ByteArray) : JsValue() {
        override fun equals(other: Any?): Boolean =
            other is Bytes && data.contentEquals(other.data)
        override fun hashCode(): Int = data.contentHashCode()
    }
}

fun JsValue.asStringOrNull(): String? = (this as? JsValue.Str)?.value
fun JsValue.asObjOrNull(): Map<String, JsValue>? = (this as? JsValue.Obj)?.entries
```

```kotlin
// JsRuntimeContractTest.kt
package top.iwesley.lyn.music.scripting

import kotlinx.coroutines.test.runTest
import kotlin.test.Test
import kotlin.test.assertEquals

class JsRuntimeContractTest {

    private fun runtime(): JsRuntime =
        JsRuntimeFactory.create(sourceId = "test", bridge = StubBridge())

    @Test
    fun `evaluate numeric expression returns Num`() = runTest {
        runtime().use { rt ->
            val r = rt.evaluate("1 + 2")
            assertEquals(JsValue.Num(3.0), r)
        }
    }

    @Test
    fun `invoke calls nested path`() = runTest {
        runtime().use { rt ->
            rt.evaluate("var api = { foo: { bar: function(x){ return x * 2; } } };")
            val r = rt.invoke("api.foo.bar", JsValue.Num(5.0))
            assertEquals(JsValue.Num(10.0), r)
        }
    }

    @Test
    fun `register exposes host function to js`() = runTest {
        runtime().use { rt ->
            rt.register("addOne") { args -> JsValue.Num((args[0] as JsValue.Num).value + 1) }
            val r = rt.evaluate("addOne(41)")
            assertEquals(JsValue.Num(42.0), r)
        }
    }
```

```kotlin
import kotlinx.coroutines.CoroutineDispatcher
import kotlinx.coroutines.Dispatchers
import kotlinx.coroutines.asCoroutineDispatcher
import kotlinx.coroutines.runBlocking
import kotlinx.coroutines.withContext
import org.graalvm.polyglot.Context
import org.graalvm.polyglot.HostAccess
import org.graalvm.polyglot.Value
import org.graalvm.polyglot.proxy.ProxyExecutable
import java.util.concurrent.Executors

actual object JsRuntimeFactory {
    actual fun create(sourceId: String, bridge: JsBridge): JsRuntime = GraalJsRuntime(sourceId, bridge)
}

private class GraalJsRuntime(
    override val sourceId: String,
    private val bridge: JsBridge,
) : JsRuntime {
    private val executor = Executors.newSingleThreadExecutor { r ->
        Thread(r, "js-$sourceId").apply { isDaemon = true }
    }
    override val dispatcher: CoroutineDispatcher = executor.asCoroutineDispatcher()

    private val context: Context = Context.newBuilder("js")
        .allowHostAccess(HostAccess.ALL)
        .option("engine.WarnInterpreterOnly", "false")
        .build()

    private val hosts = mutableMapOf<String, HostFunction>()

    init {
        // 注入 lyn 全局对象骨架；具体 Bridge 函数由 register 按需补齐
        withContext0 {
            context.eval("js", "globalThis.lyn = globalThis.lyn || {};")
        }
    }

    override suspend fun evaluate(script: String, name: String): JsValue = withContext(dispatcher) {
        withContext0 {
            val src = org.graalvm.polyglot.Source.newBuilder("js", script, name).build()
            context.eval(src).toJsValue()
        }
    }

    override suspend fun invoke(path: String, vararg args: JsValue): JsValue = withContext(dispatcher) {
        withContext0 {
            var cur: Value = context.getBindings("js")
            for (seg in path.split('.')) {
                cur = cur.getMember(seg) ?: throw MusicSourceException.ScriptRuntimeError(
                    sourceId, "path not found: $path", null
                )
            }
            val result = cur.execute(*args.map { it.toHost() }.toTypedArray())
            result.toJsValue()
        }
    }

    override fun register(name: String, host: HostFunction) {
        hosts[name] = host
        runBlocking(dispatcher) {
            val proxy = ProxyExecutable { graalArgs ->
                val parsed = graalArgs.map { it.toJsValue() }
                runBlocking { host(parsed).toHost() }
            }
            context.getBindings("js").putMember(name, proxy)
        }
    }

    override fun close() {
        try { context.close(true) } catch (_: Throwable) {}
        executor.shutdownNow()
    }

    private inline fun <T> withContext0(block: () -> T): T {
        context.enter()
        try { return block() } finally { context.leave() }
    }
}

private fun Value.toJsValue(): JsValue = when {
    isNull -> JsValue.Null
    isBoolean -> JsValue.Bool(asBoolean())
    isNumber -> JsValue.Num(asDouble())
    isString -> JsValue.Str(asString())
    hasArrayElements -> JsValue.Arr(
        (0 until arraySize).map { getArrayElement(it).toJsValue() }
    )
    hasMembers -> JsValue.Obj(
        memberKeys.associateWith { k -> getMember(k).toJsValue() }
    )
    else -> JsValue.Undefined
}

private fun JsValue.toHost(): Any? = when (this) {
    is JsValue.Null -> null
    is JsValue.Undefined -> null
    is JsValue.Bool -> value
    is JsValue.Num -> value
    is JsValue.Str -> value
    is JsValue.Bytes -> data
    is JsValue.Arr -> items.map { it.toHost() }
    is JsValue.Obj -> entries.mapValues { (_, v) -> v.toHost() }
}
```

```kotlin
            ctx.globalObject.setProperty(name) { args ->

            }
        }

        executor.shutdownNow()
```

```kotlin
import kotlinx.cinterop.ExperimentalForeignApi
import kotlinx.coroutines.CoroutineDispatcher

actual object JsRuntimeFactory {

@OptIn(ExperimentalForeignApi::class)
private class JsCoreRuntime(
    override val sourceId: String,
    private val bridge: JsBridge,
) : JsRuntime {
    @OptIn(kotlinx.coroutines.DelicateCoroutinesApi::class)
    override val dispatcher: CoroutineDispatcher =
        newSingleThreadContext("jsc-$sourceId")
    private val ctx: JSContext = JSContext()

    init {
        ctx.evaluateScript("globalThis.lyn = globalThis.lyn || {};")
    }

    override suspend fun evaluate(script: String, name: String): JsValue = withContext(dispatcher) {
        val v = ctx.evaluateScript(script) ?: return@withContext JsValue.Undefined
        v.toJsValue()
    }

    override suspend fun invoke(path: String, vararg args: JsValue): JsValue = withContext(dispatcher) {
        var cur: NativeJSValue? = ctx.globalObject
        for (seg in path.split('.')) {
            cur = cur?.objectForKeyedSubscript(seg)
                ?: throw MusicSourceException.ScriptRuntimeError(sourceId, "path not found: $path", null)
        }
        val jsArgs = args.map { it.toNative(ctx) }
        val result = cur?.callWithArguments(jsArgs) ?: return@withContext JsValue.Undefined
        result.toJsValue()
    }

    override fun register(name: String, host: HostFunction) {
        // Apple JSC 桥接块：用 objcnames / exportable block 包装
        // M0 仅 evaluate/invoke 必需，register 先占位，后续在 JsBridgeImpl.apple 中以 block → JSValue 具体实现
        throw NotImplementedError("register for Apple JSC is implemented in JsBridgeImpl (T5)")
    }

    override fun close() {
        // JSContext 随 GC 释放；单线程 context 主动 cancel
        (dispatcher as? kotlinx.coroutines.ExecutorCoroutineDispatcher)?.close()
    }
}

@OptIn(ExperimentalForeignApi::class)
private fun NativeJSValue.toJsValue(): JsValue = when {
    isNull -> JsValue.Null
    isUndefined -> JsValue.Undefined
    isBoolean -> JsValue.Bool(toBool())
    isNumber -> JsValue.Num(toDouble())
    isString -> JsValue.Str(toString())
    isArray -> {
        val n = objectForKeyedSubscript("length")?.toInt32() ?: 0
        JsValue.Arr((0 until n).map { i -> objectAtIndexedSubscript(i.toULong()).toJsValue() })
    }
    isObject -> {
        // 遍历 keys 需要借助 JSPropertyNameArray（cinterop 细节在实施时确认）
        JsValue.Obj(emptyMap())   // M0 Obj 解包走 JSON.stringify 兜底
    }
    else -> JsValue.Undefined
}

@OptIn(ExperimentalForeignApi::class)
private fun JsValue.toNative(ctx: JSContext): NativeJSValue = when (this) {
    is JsValue.Null -> NativeJSValue.valueWithNullInContext(ctx)
    is JsValue.Undefined -> NativeJSValue.valueWithUndefinedInContext(ctx)
    is JsValue.Bool -> NativeJSValue.valueWithBool(value, ctx)
    is JsValue.Num -> NativeJSValue.valueWithDouble(value, ctx)
    is JsValue.Str -> NativeJSValue.valueWithObject(value as Any, ctx)
    is JsValue.Bytes -> NativeJSValue.valueWithObject(data as Any, ctx)
    is JsValue.Arr -> NativeJSValue.valueWithObject(items.map { it.toNative(ctx) } as Any, ctx)
    is JsValue.Obj -> NativeJSValue.valueWithObject(entries.mapValues { (_, v) -> v.toNative(ctx) } as Any, ctx)
}
```

```kotlin
package top.iwesley.lyn.music.online.source

import io.ktor.client.HttpClient
import io.ktor.client.request.*
import io.ktor.client.statement.*
import io.ktor.http.*
import top.iwesley.lyn.music.scripting.JsBridge
import top.iwesley.lyn.music.scripting.JsValue

class JsBridgeImpl(
    private val http: HttpClient,
    override val platformTag: String,
    override val userAgent: String,
    private val crypto: PlatformCrypto,
) : JsBridge {

    override suspend fun request(url: String, options: Map<String, JsValue>): JsValue {
        val method = (options["method"] as? JsValue.Str)?.value ?: "GET"
        val headers = (options["headers"] as? JsValue.Obj)?.entries.orEmpty()
            .mapValues { (_, v) -> (v as? JsValue.Str)?.value.orEmpty() }
        val body = (options["body"] as? JsValue.Str)?.value

        val resp = http.request(url) {
            this.method = HttpMethod.parse(method)
            headers.forEach { (k, v) -> header(k, v) }
            header(HttpHeaders.UserAgent, headers[HttpHeaders.UserAgent] ?: userAgent)
            if (body != null) setBody(body)
        }
        val text = resp.bodyAsText()
        return JsValue.Obj(mapOf(
            "status" to JsValue.Num(resp.status.value.toDouble()),
            "statusMessage" to JsValue.Str(resp.status.description),
            "headers" to JsValue.Obj(resp.headers.toMap().mapValues { (_, v) -> JsValue.Str(v.joinToString(",")) }),
            "body" to JsValue.Str(text),
            "url" to JsValue.Str(resp.call.request.url.toString()),
        ))
    }

    override fun md5(input: ByteArray): String = crypto.md5Hex(input)
    override fun sha1(input: ByteArray): ByteArray = crypto.sha1(input)
    override fun sha256(input: ByteArray): ByteArray = crypto.sha256(input)
    override fun aesEncrypt(data: ByteArray, key: ByteArray, iv: ByteArray?, mode: String): ByteArray =
        crypto.aesEncrypt(data, key, iv, mode)
    override fun desEncrypt(data: ByteArray, key: ByteArray, iv: ByteArray?, mode: String): ByteArray =
        crypto.desEncrypt(data, key, iv, mode)
    override fun rsaEncrypt(data: ByteArray, publicKeyPem: String): ByteArray =
        crypto.rsaEncrypt(data, publicKeyPem)

    override fun base64Encode(input: ByteArray): String = crypto.base64Encode(input)
    override fun base64Decode(input: String): ByteArray = crypto.base64Decode(input)

    override fun bufferFrom(str: String, encoding: String): ByteArray =
        when (encoding.lowercase()) {
            "utf-8", "utf8" -> str.encodeToByteArray()
            "ascii" -> str.encodeToByteArray()
            "hex" -> crypto.hexDecode(str)
            "base64" -> crypto.base64Decode(str)
            else -> str.encodeToByteArray()
        }

    override fun bufferToString(bytes: ByteArray, encoding: String): String =
        when (encoding.lowercase()) {
            "utf-8", "utf8", "ascii" -> bytes.decodeToString()
            "hex" -> crypto.hexEncode(bytes)
            "base64" -> crypto.base64Encode(bytes)
            else -> bytes.decodeToString()

    override fun iconvDecode(input: ByteArray, encoding: String): String = crypto.iconvDecode(input, encoding)
```

```kotlin
// JvmPlatformCrypto（jvmMain，android 可复用）
package top.iwesley.lyn.music.online.source

import java.security.KeyFactory
import java.security.MessageDigest
import java.security.spec.X509EncodedKeySpec
import java.util.Base64
import java.util.zip.Inflater
import javax.crypto.Cipher
import javax.crypto.spec.IvParameterSpec
import javax.crypto.spec.SecretKeySpec

class JvmPlatformCrypto : PlatformCrypto {
    override fun md5Hex(input: ByteArray): String =
        MessageDigest.getInstance("MD5").digest(input).joinToString("") { "%02x".format(it) }
    override fun sha1(input: ByteArray): ByteArray = MessageDigest.getInstance("SHA-1").digest(input)
    override fun sha256(input: ByteArray): ByteArray = MessageDigest.getInstance("SHA-256").digest(input)
    override fun aesEncrypt(data: ByteArray, key: ByteArray, iv: ByteArray?, mode: String): ByteArray {
        val cipher = Cipher.getInstance("AES/${mode.uppercase()}/PKCS5Padding")
        if (iv != null) cipher.init(Cipher.ENCRYPT_MODE, SecretKeySpec(key, "AES"), IvParameterSpec(iv))
        else cipher.init(Cipher.ENCRYPT_MODE, SecretKeySpec(key, "AES"))
        return cipher.doFinal(data)
    }
    override fun desEncrypt(data: ByteArray, key: ByteArray, iv: ByteArray?, mode: String): ByteArray {
        val cipher = Cipher.getInstance("DES/${mode.uppercase()}/PKCS5Padding")
        cipher.init(Cipher.ENCRYPT_MODE, SecretKeySpec(key, "DES"), iv?.let(::IvParameterSpec))
        return cipher.doFinal(data)
    }
    override fun rsaEncrypt(data: ByteArray, publicKeyPem: String): ByteArray {
        val clean = publicKeyPem.lines().filterNot { it.startsWith("-----") }.joinToString("").trim()
        val key = KeyFactory.getInstance("RSA").generatePublic(X509EncodedKeySpec(Base64.getDecoder().decode(clean)))
        val cipher = Cipher.getInstance("RSA/ECB/PKCS1Padding")
        cipher.init(Cipher.ENCRYPT_MODE, key)
        return cipher.doFinal(data)
    }
    override fun base64Encode(input: ByteArray): String = Base64.getEncoder().encodeToString(input)
    override fun base64Decode(input: String): ByteArray = Base64.getDecoder().decode(input)
    override fun hexEncode(input: ByteArray): String = input.joinToString("") { "%02x".format(it) }
    override fun hexDecode(input: String): ByteArray =
        input.chunked(2).map { it.toInt(16).toByte() }.toByteArray()
```

```kotlin
// appleMain：用 krypto + Foundation
package top.iwesley.lyn.music.online.source

import kotlinx.cinterop.*
import platform.Foundation.*
import com.soywiz.krypto.MD5
import com.soywiz.krypto.SHA1
import com.soywiz.krypto.SHA256
import com.soywiz.krypto.AES
import com.soywiz.krypto.Padding as KryptoPadding
import platform.posix.memcpy

@OptIn(ExperimentalForeignApi::class)
class ApplePlatformCrypto : PlatformCrypto {
    override fun md5Hex(input: ByteArray): String = MD5.digest(input).hex
    override fun sha1(input: ByteArray): ByteArray = SHA1.digest(input).bytes
    override fun sha256(input: ByteArray): ByteArray = SHA256.digest(input).bytes

    override fun aesEncrypt(data: ByteArray, key: ByteArray, iv: ByteArray?, mode: String): ByteArray =
        when (mode.uppercase()) {
            "CBC" -> AES.encryptAesCbc(data, key, iv ?: ByteArray(16), KryptoPadding.PKCS7Padding)
            "ECB" -> AES.encryptAesEcb(data, key, KryptoPadding.PKCS7Padding)
            else -> AES.encryptAesCbc(data, key, iv ?: ByteArray(16), KryptoPadding.PKCS7Padding)
        }

    override fun desEncrypt(data: ByteArray, key: ByteArray, iv: ByteArray?, mode: String): ByteArray {
        // lx 五源中未见 DES 调用；M0 抛错占位，若冒烟触发再补实现
        throw NotImplementedError("DES on Apple not implemented in M0; no enabled source currently uses it")
    }

    override fun rsaEncrypt(data: ByteArray, publicKeyPem: String): ByteArray {
        // M0：wy 源 RSA 签名路径待 iOS 冒烟验证；若不触发可延至 M1
        throw NotImplementedError("RSA on Apple not implemented in M0; falls back to other sources until implemented")
    }

    override fun base64Encode(input: ByteArray): String = input.toNSData().base64EncodedStringWithOptions(0U)
    override fun base64Decode(input: String): ByteArray =
        (NSData.create(base64EncodedString = input, options = 0U) ?: return ByteArray(0)).toByteArray()

    override fun hexEncode(input: ByteArray) = input.joinToString("") { "%02x".format(it.toInt() and 0xFF) }
    override fun hexDecode(input: String) = input.chunked(2).map { it.toInt(16).toByte() }.toByteArray()

    override fun zlibInflate(input: ByteArray): ByteArray {
        // Foundation 无直接 inflate；用 NSData 的 decompressedData(using: .zlib) (iOS 13+, macOS 10.15+)
        val data = input.toNSData()
        val out = data.decompressedDataUsingAlgorithm(NSDataCompressionAlgorithmZlib, null)
            ?: return ByteArray(0)
        return out.toByteArray()
    }

    override fun iconvDecode(input: ByteArray, encoding: String): String {
        val enc = appleEncoding(encoding)
        val ns = NSString.create(data = input.toNSData(), encoding = enc)
        return ns?.toString() ?: input.decodeToString()
    }

    override fun iconvEncode(input: String, encoding: String): ByteArray {
        val enc = appleEncoding(encoding)
        val ns = (input as NSString).dataUsingEncoding(enc) ?: return input.encodeToByteArray()
        return ns.toByteArray()
    }

    private fun appleEncoding(name: String): platform.Foundation.NSStringEncoding = when (name.lowercase()) {
        "gbk", "gb18030", "gb2312" -> CFStringConvertEncodingToNSStringEncoding(kCFStringEncodingGB_18030_2000)
        "utf-8", "utf8" -> NSUTF8StringEncoding
        "ascii" -> NSASCIIStringEncoding
        else -> NSUTF8StringEncoding
    }
}

@OptIn(ExperimentalForeignApi::class)
private fun ByteArray.toNSData(): NSData =
    this.usePinned { NSData.create(bytes = it.addressOf(0), length = this.size.toULong()) }

@OptIn(ExperimentalForeignApi::class)
private fun NSData.toByteArray(): ByteArray {
    val size = this.length.toInt()
    if (size == 0) return ByteArray(0)
    val out = ByteArray(size)
    out.usePinned { pinned ->
        memcpy(pinned.addressOf(0), this.bytes, size.toULong())
    }
    return out
}
```

```kotlin
package top.iwesley.lyn.music.online.source

import kotlinx.coroutines.sync.Mutex
import kotlinx.coroutines.sync.withLock
import org.jetbrains.compose.resources.ExperimentalResourceApi
import org.jetbrains.compose.resources.Res
import top.iwesley.lyn.music.online.MusicSourceFacade
import top.iwesley.lyn.music.online.types.*
import top.iwesley.lyn.music.scripting.JsBridge
import top.iwesley.lyn.music.scripting.JsRuntime
import top.iwesley.lyn.music.scripting.JsRuntimeFactory
import top.iwesley.lyn.music.scripting.JsValue
import top.iwesley.lyn.music.scripting.MusicSourceException

@OptIn(ExperimentalResourceApi::class)
class JsMusicSource(
    private val info: SourceInfo,
    private val bridge: JsBridge,
    private val runtimeFactory: (sourceId: String, bridge: JsBridge) -> JsRuntime = JsRuntimeFactory::create,
) {
    private val loadMutex = Mutex()
    private var runtime: JsRuntime? = null

    val sourceId: String get() = info.id

    private suspend fun ensureLoaded(): JsRuntime = loadMutex.withLock {
        runtime?.let { return it }
        val rt = runtimeFactory(info.id, bridge)
        registerBridgeFunctions(rt)
        val script = Res.readBytes("files/sdk/${info.id}.js").decodeToString()
        rt.evaluate(script, "${info.id}.js")
        runtime = rt
        rt
    }

    private fun registerBridgeFunctions(rt: JsRuntime) {
        rt.register("__lyn_request") { args ->
            val url = (args[0] as JsValue.Str).value
            val opts = (args.getOrNull(1) as? JsValue.Obj)?.entries.orEmpty()
            bridge.request(url, opts)
        }
        rt.register("__lyn_md5") { args ->
            JsValue.Str(bridge.md5((args[0] as JsValue.Str).value.encodeToByteArray()))
        }
        // 其余桥接：base64Encode/Decode/aes/rsa/sha1/sha256/zlibInflate/iconvDecode/iconvEncode
        // 省略 — 照同一模式写 12 条 register。
    }

    suspend fun search(keyword: String, page: Int, limit: Int): SearchPage<OnlineSong> {
        val rt = ensureLoaded()
        val result = rt.invoke(
            "__lyn_source_${info.id}.musicSearch.search",
            JsValue.Str(keyword), JsValue.Num(page.toDouble()), JsValue.Num(limit.toDouble())
        )
        return parseSearchPage(result, info.id)
    }

    suspend fun getPlayableUrl(songmid: String, quality: Quality): PlayableUrl {
        val rt = ensureLoaded()
        val musicInfo = JsValue.Obj(mapOf(
            "songmid" to JsValue.Str(songmid),
            "source" to JsValue.Str(info.id),
        ))
        val result = rt.invoke("__lyn_source_${info.id}.getMusicUrl", musicInfo, JsValue.Str(quality.lxKey))
        val url = (result as? JsValue.Obj)?.entries?.get("url")?.let { (it as? JsValue.Str)?.value }
            ?: throw MusicSourceException.Parse(info.id, null)
        return PlayableUrl(url, quality, kotlin.time.Clock.System.now())
    }

    suspend fun getLyric(songmid: String): OnlineLyric {
        val rt = ensureLoaded()
        val musicInfo = JsValue.Obj(mapOf("songmid" to JsValue.Str(songmid), "source" to JsValue.Str(info.id)))
        val r = rt.invoke("__lyn_source_${info.id}.getLyric", musicInfo)
        val o = r as? JsValue.Obj ?: throw MusicSourceException.Parse(info.id, null)
        return OnlineLyric(
            original = (o.entries["lyric"] as? JsValue.Str)?.value.orEmpty(),
            translation = (o.entries["tlyric"] as? JsValue.Str)?.value,
            enhanced = (o.entries["lxlyric"] as? JsValue.Str)?.value,
        )
    }

    suspend fun getPic(songmid: String): String {
        val rt = ensureLoaded()
        val musicInfo = JsValue.Obj(mapOf("songmid" to JsValue.Str(songmid), "source" to JsValue.Str(info.id)))
        val r = rt.invoke("__lyn_source_${info.id}.getPic", musicInfo)
        return (r as? JsValue.Str)?.value ?: throw MusicSourceException.Parse(info.id, null)
    }

    fun close() { runtime?.close(); runtime = null }
}

private fun parseSearchPage(result: JsValue, sourceId: String): SearchPage<OnlineSong> {
    val obj = (result as? JsValue.Obj)?.entries ?: throw MusicSourceException.Parse(sourceId, null)
    val list = (obj["list"] as? JsValue.Arr)?.items.orEmpty().mapNotNull { it.toOnlineSong(sourceId) }
    val total = (obj["total"] as? JsValue.Num)?.value?.toInt() ?: list.size
    val allPage = (obj["allPage"] as? JsValue.Num)?.value?.toInt() ?: 1
    return SearchPage(items = list, page = 1, totalPages = allPage, totalItems = total, sourceId = sourceId)
}

private fun JsValue.toOnlineSong(sourceId: String): OnlineSong? {
    val o = (this as? JsValue.Obj)?.entries ?: return null
    fun s(k: String) = (o[k] as? JsValue.Str)?.value
    val songmid = s("songmid") ?: return null
    val name = s("name") ?: return null
    val singer = s("singer") ?: return null
    val interval = s("interval").orEmpty()   // "mm:ss"
    val intervalSec = interval.split(":").mapNotNull { it.toIntOrNull() }
        .let { if (it.size == 2) it[0] * 60 + it[1] else 0 }
    val availableLxKeys = ((o["types"] as? JsValue.Arr)?.items.orEmpty())
        .mapNotNull { ((it as? JsValue.Obj)?.entries?.get("type") as? JsValue.Str)?.value }
    val qualities = availableLxKeys.mapNotNull { Quality.fromLxKey(it) }.distinct()
    return OnlineSong(
        id = OnlineMusicId(sourceId, songmid, s("albumId")),
        name = name, singer = singer,
        album = s("albumName"), albumId = s("albumId"),
        intervalSeconds = intervalSec,
        coverUrl = s("img"),
        availableQualities = qualities,
        defaultQuality = qualities.firstOrNull() ?: Quality.K320,
    )
}
```

```kotlin
package top.iwesley.lyn.music.online.cache

class OnlineMemoryCache<V : Any>(private val capacity: Int = 128) {
    private val map = object : LinkedHashMap<String, V>(capacity, 0.75f, true) {
        override fun removeEldestEntry(eldest: Map.Entry<String, V>?): Boolean = size > capacity
    }

    @Synchronized
    fun get(key: String): V? = map[key]

    @Synchronized
    fun put(key: String, value: V) { map[key] = value }

    @Synchronized
    fun invalidate(key: String) { map.remove(key) }

    @Synchronized
    fun clear() { map.clear() }
}
```

```kotlin
package top.iwesley.lyn.music.online.store

import kotlinx.coroutines.CoroutineScope
import top.iwesley.lyn.music.core.mvi.BaseStore
import top.iwesley.lyn.music.online.repository.OnlineMusicRepository
import top.iwesley.lyn.music.online.types.*
import top.iwesley.lyn.music.scripting.MusicSourceException

data class OnlineSearchState(
    val query: String = "",
    val activeSource: String = "kw",
    val availableSources: List<SourceInfo> = SourceManifest.enabled,
    val perSourceResults: Map<String, SearchPage<OnlineSong>> = emptyMap(),
    val loadingBySource: Map<String, Boolean> = emptyMap(),
    val errorBySource: Map<String, String> = emptyMap(),
)

sealed interface OnlineSearchIntent {
    data class QueryChanged(val q: String) : OnlineSearchIntent
    data class SourceSelected(val sourceId: String) : OnlineSearchIntent
    data object Retry : OnlineSearchIntent
}

sealed interface OnlineSearchEffect {
    data class ShowError(val sourceId: String, val message: String) : OnlineSearchEffect
}

class OnlineSearchStore(
    private val repository: OnlineMusicRepository,
    scope: CoroutineScope,
) : BaseStore<OnlineSearchState, OnlineSearchIntent, OnlineSearchEffect>(OnlineSearchState(), scope) {

    override suspend fun handleIntent(intent: OnlineSearchIntent) {
        when (intent) {
            is OnlineSearchIntent.QueryChanged -> {
                updateState { it.copy(query = intent.q) }
                runSearch(state.value.activeSource, intent.q)
            }
            is OnlineSearchIntent.SourceSelected -> {
                updateState { it.copy(activeSource = intent.sourceId) }
                val q = state.value.query
                if (q.isNotBlank()) runSearch(intent.sourceId, q)
            }
            OnlineSearchIntent.Retry -> runSearch(state.value.activeSource, state.value.query)
        }
    }

    private suspend fun runSearch(sourceId: String, q: String) {
        if (q.isBlank()) return
        updateState { it.copy(loadingBySource = it.loadingBySource + (sourceId to true),
            errorBySource = it.errorBySource - sourceId) }
        try {
            val page = repository.search(sourceId, q)
            updateState { it.copy(
                perSourceResults = it.perSourceResults + (sourceId to page),
                loadingBySource = it.loadingBySource + (sourceId to false),
            )}
        } catch (e: MusicSourceException) {
            updateState { it.copy(
                loadingBySource = it.loadingBySource + (sourceId to false),
                errorBySource = it.errorBySource + (sourceId to (e.message ?: "失败")),
            )}
            emitEffect(OnlineSearchEffect.ShowError(sourceId, e.message ?: "失败"))
        }
    }
}
```

```kotlin
package top.iwesley.lyn.music.online.resolve

import top.iwesley.lyn.music.core.model.DiagnosticLogLevel
import top.iwesley.lyn.music.core.model.GlobalDiagnosticLogger
import top.iwesley.lyn.music.online.diagnostics.OnlineLogTags
import top.iwesley.lyn.music.online.repository.OnlineMusicRepository
import top.iwesley.lyn.music.online.types.*
import top.iwesley.lyn.music.scripting.MusicSourceException

data class ResolvedUrl(val url: String, val quality: Quality, val sourceId: String)

interface SongUrlResolver {
    /** 输入在线歌曲 ID + 默认音质；返回可播放 URL；失败抛 [MusicSourceException]。 */
    suspend fun resolve(id: OnlineMusicId, preferredQuality: Quality, songContext: OnlineSong? = null): ResolvedUrl
}

class DefaultSongUrlResolver(
    private val repository: OnlineMusicRepository,
    private val findMusic: FindMusicM0,
) : SongUrlResolver {
    override suspend fun resolve(
        id: OnlineMusicId,
        preferredQuality: Quality,
        songContext: OnlineSong?,
    ): ResolvedUrl {
        val degrade = Quality.degradationOrder.dropWhile { it != preferredQuality }
            .ifEmpty { Quality.degradationOrder }

        // 同源降级
        for (q in degrade) {
            try {
                val u = repository.getPlayableUrl(id, q)
                log(id.source, "url-ok q=${q.lxKey}")
                return ResolvedUrl(u.url, q, id.source)
            } catch (e: MusicSourceException) {
                log(id.source, "url-fail q=${q.lxKey}: ${e.message}", DiagnosticLogLevel.WARN, e)
                continue
            }
        }

        // 跨源 findMusic（M0 仅在有 songContext 时执行）
        if (songContext != null) {
            val alt = findMusic.find(songContext, excludeSource = id.source)
            if (alt != null) {
                try {
                    val u = repository.getPlayableUrl(alt.id, alt.defaultQuality)
                    log(id.source, "cross-source-fallback source=${alt.id.source} q=${alt.defaultQuality.lxKey}")
                    return ResolvedUrl(u.url, alt.defaultQuality, alt.id.source)
                } catch (e: MusicSourceException) {
                    log(alt.id.source, "cross-fail: ${e.message}", DiagnosticLogLevel.ERROR, e)
                }
            }
        }
        throw MusicSourceException.UrlExpired(id.source)
    }

    private fun log(sourceId: String, message: String,
                    level: DiagnosticLogLevel = DiagnosticLogLevel.DEBUG, t: Throwable? = null) {
        GlobalDiagnosticLogger.log(level, OnlineLogTags.source(sourceId), message, t)
    }
}
```

- [ ] **Step 2: 写 `FindMusicM0.kt`（2-pass 简化版）**

```kotlin
package top.iwesley.lyn.music.online.resolve

import top.iwesley.lyn.music.online.repository.OnlineMusicRepository
import top.iwesley.lyn.music.online.types.OnlineSong
import top.iwesley.lyn.music.online.types.SourceManifest

class FindMusicM0(
    private val repository: OnlineMusicRepository,
    private val durationToleranceSec: Int = 3,
) {
    suspend fun find(target: OnlineSong, excludeSource: String): OnlineSong? {
        val tName = normalize(target.name)
        val tSinger = normalize(target.singer)
        val keyword = "${target.name} ${target.singer}".trim()

        for (src in SourceManifest.enabled) {
            if (src.id == excludeSource) continue
            val page = runCatching { repository.search(src.id, keyword, page = 1, limit = 20) }.getOrNull() ?: continue

            // Pass 1: 严格匹配 name + singer + duration
            page.items.firstOrNull { s ->
                normalize(s.name) == tName &&
                    normalize(s.singer) == tSinger &&
                    kotlin.math.abs(s.intervalSeconds - target.intervalSeconds) <= durationToleranceSec
            }?.let { return it }

            // Pass 2: 宽松：name 精确 + singer 任一主歌手出现
            val targetSingers = tSinger.split(Regex("[,、/&]")).filter { it.isNotBlank() }.toSet()
            page.items.firstOrNull { s ->
                normalize(s.name) == tName &&
                    normalize(s.singer).split(Regex("[,、/&]")).any { it.trim() in targetSingers }
            }?.let { return it }
        }
        return null
    }

    private fun normalize(s: String): String =
        s.trim().lowercase().replace(Regex("[\\s()（）\\[\\]【】]"), "")
}
```

```kotlin
private suspend fun loadGatewaySafely(track: Track, playWhenReady: Boolean, startPositionMs: Long, loadToken: PlaybackLoadToken) {
    val resolved = if (isOnlineTrack(track)) {
        try {
            val parsed = parseOnlineLocator(track.mediaLocator)   // "online-lazy://{source}/{songmid}?q={quality}"
            val resolved = songUrlResolver.resolve(
                id = OnlineMusicId(parsed.source, parsed.songmid),
                preferredQuality = parsed.quality,
                songContext = null,   // M0: 暂不传；T10 的 OnlineSongToTrack 会在 extras 里塞
            )
            track.copy(mediaLocator = resolved.url)
        } catch (e: MusicSourceException) {
            GlobalDiagnosticLogger.log(DiagnosticLogLevel.ERROR,
                OnlineLogTags.RESOLVER, "resolve failed for ${track.sourceId}: ${e.message}", e)
            throw e
        }
    } else track
    gateway.load(resolved, playWhenReady, startPositionMs, loadToken)
}

private fun isOnlineTrack(t: Track): Boolean =
    t.sourceId.startsWith("online-") || t.mediaLocator.startsWith("online-lazy://")

private data class OnlineLocator(val source: String, val songmid: String, val quality: Quality)

private fun parseOnlineLocator(loc: String): OnlineLocator {
    // 格式：online-lazy://kw/abc123?q=320k
    val uri = loc.removePrefix("online-lazy://")
    val (path, qs) = uri.split('?', limit = 2).let { (it.getOrNull(0).orEmpty()) to (it.getOrNull(1).orEmpty()) }
    val (source, songmid) = path.split('/', limit = 2).let { (it.getOrNull(0).orEmpty()) to (it.getOrNull(1).orEmpty()) }
    val qKey = qs.split('&').mapNotNull { it.split('=').takeIf { p -> p.size == 2 } }
        .firstOrNull { it[0] == "q" }?.get(1) ?: "320k"
    val quality = Quality.fromLxKey(qKey) ?: Quality.K320
    return OnlineLocator(source, songmid, quality)
}
```

```kotlin
fun formatJvmCrashReport(threadName: String, throwable: Throwable): String = buildString {
    // ... 原有格式
    val online = unwrapOnline(throwable)
    if (online != null) {
        appendLine()
        appendLine("=== Online Music Context ===")
        appendLine("source: ${online.sourceId}")
        if (online is MusicSourceException.ScriptRuntimeError) appendLine("jsStack: ${online.jsStack}")
        if (online is MusicSourceException.Timeout) appendLine("stage: ${online.stage}")
        if (online is MusicSourceException.Network) appendLine("http: ${online.code}")
    }
}

private fun unwrapOnline(t: Throwable?): MusicSourceException? {
    var cur: Throwable? = t
    while (cur != null) { if (cur is MusicSourceException) return cur; cur = cur.cause }
    return null
}
```
